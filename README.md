## Usage

### 0. clone the repository using command
```bash
git clone https://github.com/idehweb/use-nodeeweb.git
```
or manually download

### 1. in the root of project, in command line or terminal:
```bash
npm install
```
or using yarn:
```bash
yarn
```
### 2. now open http://localhost:3000
```bash
npm start
```
or
```bash
yarn start
```
or
```bash
node index.mjs
```
### 3. after that .env.local will be created

after you start server for first time, .env.local will be created, you can change configurations in it and restart server
